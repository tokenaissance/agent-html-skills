# Agent HTML Skills

A Claude Code plugin providing 20+ specialized HTML-output skills. Each skill transforms
a specific type of AI response into an interactive, visually rich HTML artifact.

## Skills

| Skill | Use When |
|-------|----------|
| `html-spec-planning` | Specs, RFCs, implementation plans |
| `html-code-review` | PR explainers, refactor risk maps, codebase tours |
| `html-design-prototypes` | Component design, animation tuning, design systems |
| `html-research-reports` | Multi-source research synthesis, status reports |
| `html-throwaway-editor` | One-off editors with copy-as-prompt export |
| `html-interactive-playground` | Sliders/knobs for parameter tuning |
| `html-brainstorm-grid` | N-variant comparison grids with tradeoff labels |
| `html-svg-diagrams` | Flowcharts, sequence diagrams, state machines |
| `html-slideshow-deck` | Keyboard-navigable presentation decks |
| `html-design-tokens` | Color/type/spacing token showcases |
| `html-architecture-diagrams` | System maps, deployment topologies |
| `html-data-explorer` | Filterable tables, faceted search, log viewers |
| `html-mind-map` | Hierarchical mind maps |
| `html-comparison-matrix` | Side-by-side comparisons |
| `html-timeline-roadmap` | Timeline and roadmap visualizations |
| `html-erd-explorer` | Entity-relationship diagrams |

## Installation

```bash
npx skills add tokenaissance/agent-html-skills
```

Or via Claude Code plugin: add `tokenaissance/agent-html-skills` to your plugins list.
